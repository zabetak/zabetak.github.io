CliqueSquare: massively parallel RDF data management

https://team.inria.fr/oak/projects/cliquesquare/

CliqueSquare is a system for storing and querying large RDF graphs relying on Hadoopâ€™s distributed file system (HDFS) and Hadoopâ€™s MapReduce open-source implementation. It provides a novel partitioning and storage scheme that permits 1-level joins to be evaluated locally using efficient map-only joins.
